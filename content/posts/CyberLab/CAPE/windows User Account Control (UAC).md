### UAC setting

\*\*1.Open the UAC Settings：\*\*Requires local administrator rights。

Press the Windows key, type **UAC**, and select **Change User Account Control settings** from the search results.

![Pasted image 20260724200713.png](/ob/Pasted%20image%2020260724200713.png)

**2.Adjust the Slider：**

Move the vertical slider to set the desired notification threshold (details below).

![Pasted image 20260724200739.png](/ob/Pasted%20image%2020260724200739.png)

**3.Save Changes：**

Click **OK**. You will be prompted to confirm this change on the secure desktop.
